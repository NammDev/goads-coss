## Phase 02: Section Blocks (14 Sections)

**Priority:** P1 | **Status:** in-progress | **Effort:** 8h
**Depends on:** Phase 01 complete
**Source:** 80% almarmedia.com / 20% GoAds original

### Workflow Per Section (MANDATORY)

For every section:
1. Use shadcn studio MCP `get-blocks-metadata` to browse available variants in the category
2. Pick best-fit variant based on content needs (documented per section below)
3. Install via `/cui` MCP workflow: `npx shadcn@latest add @ss-blocks/{block-name}`
4. Swap all content (text, hrefs, images) per content spec below
5. Strip block container classes â†’ apply goads `container` (`mx-auto max-w-[1416px] px-4 lg:px-6`)
6. Remove any block-internal header/navbar/footer elements
7. Verify renders correctly at mobile (375px), tablet (768px), desktop (1440px)

### CRITICAL RULE: Respect GoAds Layout

**ALWAYS adapt shadcn studio blocks to match goads layout, NOT the other way around.**
Blocks are templates â€” goads layout (grid frame lines, container, corner dots) is the source of truth.

### Layout Adaptation Checklist (Apply to EVERY block)

- [ ] Remove `max-w-7xl`, `border-x`, `sm:px-6`, `lg:px-8` from block wrapper
- [ ] Add goads `container` class (`mx-auto max-w-[1416px] px-4 lg:px-6`) to outermost section div
- [ ] Add `border-b border-border/60` as section separator (except last section)
- [ ] No block-internal nav, header, or footer elements
- [ ] Background color set at section level, not block wrapper level
- [ ] Verify section aligns with vertical grid frame lines in layout.tsx
- [ ] No double containers â€” one `container` wrapper per section max
