# Portable Skills

Skills stored here are version-controlled and can be installed on any machine.

## Install all skills to global Claude

Run this on any machine after cloning the repo:

```bash
cp -r skills/* ~/.claude/skills/
```

## Install a specific skill

```bash
cp -r skills/clone-foreplay ~/.claude/skills/clone-foreplay
```

After copying, restart Claude Code to pick up the new skill.
